# Student Database Management System

## Description
This project is a Student Database Management System that manages student information and attendance using database concepts. The system includes separate login access for Admin, Faculty, and Students.

## Technologies Used
- HTML
- DBMS / SQL
- Basic Web Development

## System Roles

### Admin
- Login access for admin
- Can add, update, and manage student records
- Can modify student details in the database

### Faculty
- Login access for faculty members
- Can mark and update student attendance

### Student
- Login access for students
- Can view their personal details
- Can check their attendance records

## Features
- Role-based login system
- Student information management
- Attendance tracking
- Database storage of records
